# 🛍️ ClickShop — E-Commerce App

A fully functional e-commerce web application built with **Next.js 15** (App Router) and **TypeScript**, consuming a RESTful API. The app features a complete shopping experience including product browsing, cart management, favorites, checkout, and order tracking.

## ⚙️ Tech Stack

| Technology | Usage |
|------------|-------|
| Next.js 15 (App Router) | Framework |
| TypeScript | Type Safety |
| Redux Toolkit | State Management |
| MUI v6 (Material UI) | UI Components |
| Axios | HTTP Client |
| Formik + Yup | Form Validation |
| React Hot Toast | Notifications |
| Swiper.js | Slider |
| Next.js Middleware | Route Protection |

---

## ✅ Features

- Authentication** — Register / Login / Logout with JWT token
- Protected Routes** — via Next.js Middleware using Cookies
- Products** — Browse, Filter by Category & Brand, Search
- Product Details** — Full details + Customer Reviews
- Cart** — Add / Remove / Update Quantity
- Favorites** — Save & manage favorite products
- Checkout** — Cash on Delivery & Online Payment
- Orders** — View order history
- Reviews** — Add & view product reviews
- Responsive Design** — Mobile & Desktop friendly
